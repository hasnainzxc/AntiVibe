# Feature: Exec Report Generator

**Purpose:** Generate markdown + JSON "FixIt receipt" from normalized findings. Output emailable + dashboard-viewable.
**Wave:** 5  **Owner task:** 31  **Status:** pending

## Public API
```python
class ReportGenerator:
    def generate(self, *, normalized_findings: list[Finding], scan_result: ScanResult) -> tuple[str, dict]:
        """Returns (markdown, json_dict)."""
```

## Internal flow
1. Sort findings by severity (critical → high → medium → low → info)
2. For each: section w/ title, location (file:line), evidence (masked curl), remediation patch (diff), CVSS-ish score
3. Top: summary table (X critical, Y high, Z medium, ...)
4. Markdown template includes footer "Generated by AntiVibe — audit your vibe-coded apps at antivibe.app"
5. JSON wraps same data as `consultants` compatible shape per `Report` schema in `docs/system-design.md`

## Acceptance criteria
- [ ] ≥1 critical → markdown has emoji 🔴 + summary table top
- [ ] No raw tokens in evidence (masked)
- [ ] Renderable in Next.js `markdown-to-jsx` dashboard view
- [ ] JSON matches `ScanResult` TS type
