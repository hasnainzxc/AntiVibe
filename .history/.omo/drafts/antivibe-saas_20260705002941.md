# Draft: AntiVibe SaaS

## Core Objective
SaaS that audits AI-generated ("vibecoded") apps via GitHub URL. 3 tiers:
1. Static Intelligence Engine (AST + regex + LLM semantic parse) — secret detection, config flaw logic (Firestore rules, IAM policies).
2. Ephemeral Sandbox Spin-up (microVMs, mock DBs, Firebase emulators) — isolated app runtime.
3. Autonomous Fuzzing Agent (JWT forgery, BOLA/IDOR test, no-stop on 403/404) — deductive feedback loop like human pentester.

## User-Stated Requirements (from blueprint)
- Input: GitHub URL → output: exec summary of critical/non-critical fixes w/ remediation code snippets + auto-PR.
- Tier 1: zero-false-positive secret detection; logic flaw reporting (e.g., open Firestore rules vs expected auth context).
- Tier 2: AWS Firecracker or Fly.io microVMs, <120ms boot; DB mocking w/ synthetic tenant profiles (User_A student, User_B admin).
- Tier 3: route extraction from code → JWT forge pipeline (2 dummy users) → cognitive fuzzing loop; swap URL params, cross-tenant; never stop on 403/404 — pivot to adjacent resources.
- Dual-model arch: commercial LLM for structural extraction (no guardrail trips) + self-hosted OSS model (DeepSeek-Coder-33B / Llama-3-70B via vLLM) for aggressive fuzzing (no refusals).
- Auto-PR generation with fixed config rules.
- YC-level pitch differentiators: AI-native speed (5-min scan post-push), context-aware business-logic flaws, auto code patching.

## Metis Guardrails (incorporated)
- $0.50/scan max cost (Fly Machine + LLM tokens). Circuit-breaker at 10min runtime.
- Sandbox egress = DENY ALL except localhost (app-under-test).
- Repo clone = shallow (--depth 1), no LFS, max 500MB.
- Block postinstall scripts in sandbox (npm install --ignore-scripts).
- GitHub webhook = HMAC-SHA256 signature verification mandatory.
- Auto-PR = human review required (NEVER auto-merge).
- LLM input sanitization = strip secrets/PII before API call.
- Fly Machine auto-destroy post-scan (no lingering).
- Rate limit = 1 scan/hour/IP on free tier.
- Email verification required before free scan.
- Latency targets: Tier1 p95 <5min, Tier2+3 p95 <15min.
- Secret detector FP rate <5% (measure on 100 repos).
- Stack detector accuracy >90% (measure on 50 repos).
- LLM token usage <100K tokens/scan (measure on 10 scans).
